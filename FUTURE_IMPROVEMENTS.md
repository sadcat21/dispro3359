## قائمة التعديلات المطلوبة

### 1. صفحة مخزون الفرع (WarehouseStock)
- **مشكلة**: لا يُسمح بالتمرير (Scroll) لقائمة المنتجات — يجب إصلاح ذلك للسماح بعرض كافة المنتجات.
- **تحسين**: يجب التفريق بوضوح بين قسم **رصيد المخزون** (مخزون الفرع) وقسم **رصيد العمال** (مخزون العمال) داخل نفس الصفحة بفواصل بصرية واضحة.

### 2. نظام الرسائل والإشعارات
- **إشعارات Push**: تفعيل نظام الإشعارات باستخدام Firebase Cloud Messaging (FCM) لإرسال تنبيهات فورية للعمال والمدراء.
- **فتح تطبيق الرسائل**: إضافة أزرار لفتح تطبيق SMS مباشرة مع نص جاهز (مثل: إشعار التوصيل، تذكير الدين) باستخدام روابط `sms:`.
- **إرسال SMS تلقائي**: التكامل مع خدمة مثل Twilio أو Vonage لإرسال رسائل نصية تلقائية للعملاء (مدفوعة).
- **WhatsApp Business API**: إمكانية إرسال رسائل واتساب تلقائية للعملاء عبر API رسمي.
- **أصوات التنبيه**: تفعيل أصوات مخصصة للإشعارات المختلفة (طلب جديد، تحصيل دين، إلخ).

### 3. تطبيق أندرويد (APK)
- **Capacitor**: تم إعداد Capacitor لبناء تطبيق أندرويد حقيقي (APK) — يحتاج المستخدم لتشغيل الأوامر محلياً أو عبر GitHub Codespaces.
- **بديل**: يمكن استخدام التطبيق كـ PWA (تطبيق ويب قابل للتثبيت) مباشرة من المتصفح بدون بناء APK.
